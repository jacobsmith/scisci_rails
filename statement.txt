SciSci is a note-taking application designed for middle-school and high-school students.

SciSci allows group collaboration on projects.
  User owns project - only owner can delete project
    project has_many collaborators
      collaborators have read, read+write, or read+write+delete access


SciSci allows teachers to set a group of students and form them into groups and allow student access.
